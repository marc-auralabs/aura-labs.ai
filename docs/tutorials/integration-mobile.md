## Overview

Deploy Scout agents in native iOS and Android applications for seamless mobile shopping experiences. This guide covers integrating the AURA Scout SDK with mobile platforms, handling mobile-specific constraints, and optimizing for on-the-go transactions.

- Integrating AURA Scout SDK into iOS and Android apps
- Handling mobile-specific permission and notification flows
- Optimizing Scout search and negotiation for mobile networks
- Managing Scout background operations and foreground updates
- Implementing push notifications for transaction status
- Handling offline Scout mode and synchronization
- Mobile-specific best practices and performance optimization

## Prerequisites

- iOS and/or Android development environment
- AURA developer account
- Basic understanding of Scout
