## Overview

Embed Scout agents directly into your React application to enable automated purchasing on behalf of your users. This guide covers integrating the AURA Scout SDK with React components and managing Scout lifecycle in a frontend application.

## What You'll Learn

- Installing and configuring the AURA Scout SDK for React
- Creating Scout instances within React components
- Setting up Scout preferences and constraints
- Displaying Scout search results and negotiation status
- Handling user approvals for Scout transactions
- Managing Scout context and state in React
- Error handling and event monitoring

## Prerequisites

- React application
- AURA developer account
- Basic understanding of Scout

## Estimated Time

~30 minutes
